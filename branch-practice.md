# 🎯 Git分支实战练习记录

## 练习目标
学习Git分支的创建、切换、合并和删除

## 练习步骤

### 1. 创建分支
```bash
git checkout -b feature-practice
```
✅ 已完成 - 创建并切换到 feature-practice 分支

### 2. 在新分支上工作
- 创建这个文件
- 添加练习内容

### 3. 提交更改
```bash
git add .
git commit -m "练习：分支管理"
```

### 4. 切换回主分支
```bash
git checkout main
```

### 5. 合并分支
```bash
git merge feature-practice
```

### 6. 删除分支
```bash
git branch -d feature-practice
```

### 7. 推送到远程
```bash
git push
```

---

## 学到的知识点

1. ✅ 分支可以让我们安全地开发新功能
2. ✅ 在分支上的修改不会影响主分支
3. ✅ 合并分支会将修改应用到主分支
4. ✅ 完成后可以删除不需要的分支
